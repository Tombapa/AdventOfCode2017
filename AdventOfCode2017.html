DAY 4: HIGH-ENTROPY PASSPHRASES Dec 4, 2017

To ensure security, a valid passphrase must contain no duplicate words.

For this one, I had to learn a bit of Node.js for loading the input file. Then, input is split into lines i.e. word-based String arrays, and each array is checked for duplicated words.

node day4part1.js
    validatePassphrase(1): It's true to say this one is valid.
    validatePassphrase(2): It's true to say this one is valid.
        ** 'caibh', 'nfuk', 'kfnu', **
        ** 'llfdbz', 'uxjty', 'yxjut', 'jcea' **
    validatePassphrase(3): It's true to say this one is valid.
    ...
    validatePassphrase(7): It's true to say this one is valid.
        ** I've found a match! **
        ** 'bqs', 'bqs', 'dbutvgf', 'mmzb', 'izpyud', **
        ** 'rap', 'izpyud', 'xlzeb', 'mnj', 'hjncs' **
    validatePassphrase(8): It's false to say this one is valid.
    validatePassphrase(9): It's true to say this one is valid.
    ...
Whew, I think I'm done! I've validated 512 passphrases.


Part 1 completed. The correct answer is 466[hover].
Part Two

Now, a valid passphrase must contain no two words that are anagrams of each other - that is, a passphrase is invalid if any word's letters can be rearranged to form any other word in the passphrase.

As a new feature, each String is splitted and sorted into char array before duplicate check.

node day4part2.js
    validatePassphrase(1): It's true to say this one is valid.
    validatePassphrase(2): It's true to say this one is valid.
        ** I've found a match! **
        ** I've found a match at indexes [1] and [2]! **
        ** strA==fknu, strB==fknu **
    validatePassphrase(3): It's false to say this one is valid.
    validatePassphrase(4): It's true to say this one is valid.
    ...
    validatePassphrase(7): It's true to say this one is valid.
        ** I've found a match at indexes [0] and [1]! **
        ** strA==bqs, strB==bqs **
    validatePassphrase(8): It's false to say this one is valid.
    ...
Whew, I think I'm done! I've validated 512 passphrases.


Part 2 completed. The correct answer is 251[hover].
